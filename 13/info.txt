CLI для скачивания видео с ютуба по ссылке
1)	Продумать логику приложения
Когда запускается приложение необходимо иметь ссылку на видео
Входные данные:
	Ссылка на видео
	Указание папки куда будет помещано видео
Выходные данные:
	Полный путь куда поместился файл
2)	Что нужно чтобы сделать проект
2.1) изучить как скачивать сторонние библиотеки
2.2) создать файл который хранит ссылки на библиотеки
go mod init
2.3) go get название_библиотеки
2.4) go run название_файла
Все устанволеннные библиотеки будут проверены и докачают если были обновления

go mod init
go get github.com/faiface/beep
go run 1.go
