структирование данных

array
1) элементы одного типа если не interface
2) каждый элемент имеет свой индекс
3) имеет фиксированную длину(не динамический)

slice
1) элементы одного типа если не interface
2) каждый элемент имеет свой индекс
3) болеее динамический и имеет способность добавлять новые элементы

map -> словарь
1) элементы одного типа если не interface
2) каждому значению нужно самостоятельно создвать индекс
3) абсолютно динамический

индекс -> ключ
значение -> значение
